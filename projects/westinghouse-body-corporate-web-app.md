# Westinghouse Body Corporate Web App

## Problem Solved

Body corporate administration often involves records, requests, communication, documents, and recurring administrative tasks spread across email and spreadsheets. This project organizes those workflows into a web application with structured data capture and clearer operational tracking.

## Business Context

The app supports administrative visibility and repeatable handling of body corporate information. It is focused on practical internal operations rather than a public-facing marketing site.

## Tech Stack

- Python
- Django
- PostgreSQL
- Django forms and admin patterns
- Authentication and role-aware access
- Reporting-oriented data structures

## Key Features

- Structured records for administrative entities and workflow items
- Form-based data capture with validation
- User authentication and controlled access to operational views
- Status tracking for requests or administrative processes
- Reporting-friendly database design
- Audit-aware handling of important records

## Data / Automation Work Involved

- Converted informal administrative tracking into structured database records
- Applied validation and consistent forms to reduce data quality issues
- Created workflows that support filtering, review, and reporting
- Reduced reliance on scattered spreadsheets or email-only tracking
- Prepared data for operational reporting and follow-up

## Testing and Quality Notes

- Tested core Django views, forms, permissions, and model behavior where applicable
- Reviewed administrative workflows for usability and data consistency
- Avoided exposing resident details, building-specific records, documents, or private communications
- Kept deployment and environment secrets out of repository materials

## What This Demonstrates To Employers

- Django web application development for real administrative operations
- Database-backed workflow design using PostgreSQL
- Understanding of validation, access control, and internal reporting needs
- Ability to build practical systems for non-technical operational users
